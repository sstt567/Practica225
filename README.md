Проект "Магазин"

## Требования
- Java 17+
- IntelliJ IDEA Community Edition

## Как запустить
1. Откройте проект в IntelliJ IDEA.
2. Убедитесь, что установлен JDK 17.
3. Создайте файлы `Product.java`, `Cart.java`, `Main.java` 
4. Запустите класс `Main` 

Программа создаёт три товара, добавляет их в корзину, показывает итог, 
удаляет товар "Мышка" и снова показывает корзину.
